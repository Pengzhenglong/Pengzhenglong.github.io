判断回文字符串

方法一： 利用Array的reverse方法

方法二： 回文字符串是对称的

非空字符串删除一个元素，判断能否成为回文字符串

描述：给定一个非空字符串 s，最多删除一个字符。判断是否能成为回文字符串。
function isTransHuiwenStr(str) {
const arr = str.split(‘’)
for(let i = 0; i < str.length; i++) {
const copy = […arr]
copy.splice(i, 1)
if (copy.reverse().join(‘’) === copy.join(‘’)) {
return true
}
}
return false
}
